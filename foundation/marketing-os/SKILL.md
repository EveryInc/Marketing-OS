# Marketing OS

The foundational marketing operating system that underlies all Every skills — the voice, strategy, and craft OS.

## When to invoke

- When building or auditing any new skill for method alignment
- When onboarding a new brand into the Every system
- When resolving a conflict between brand-voice and positioning guidance

## Dependencies

None — this is the root dependency. All other skills load this skill.
